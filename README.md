# Image Retrieval Project

## Dataset

For this project, we use the **Caltech-101 dataset**, which contains images of multiple object categories with several instances per class.  
This dataset is used to test the ability to retrieve similar images.

You can download the dataset from the official source: [Caltech-101 Dataset](http://www.vision.caltech.edu/Image_Datasets/Caltech101/)

> Note: The dataset is not included in this repository due to size constraints. Please download it separately and place it in the `data/` folder.
